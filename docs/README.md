# Приложение "Путевой Компас"

## Обзор

Это приложение на Next.js, разработанное как AI-помощник для планирования путешествий. Оно помогает пользователям создавать маршруты поездок и получать рекомендации по турам, жилью, аренде авто и развлечениям от искусственного интеллекта. Владельцы бизнеса могут добавлять и управлять своими предложениями.

## Технологический стек

*   **Фреймворк**: Next.js (с App Router)
*   **Язык**: TypeScript
*   **UI**: React
*   **Стилизация**: Tailwind CSS
*   **UI Компоненты**: ShadCN UI
*   **База данных**: Firestore
*   **Аутентификация**: Firebase Authentication
*   **AI**: Google Genkit
*   **Иконки**: Lucide React
*   **Карты**: Google Maps React (@vis.gl/react-google-maps)
*   **Формы**: React Hook Form и Zod

## Функционал

### 1. Планирование путешествий
*   **Создание маршрутов**: Пользователи могут создавать новые маршруты путешествий, указывая название, пункт назначения и даты поездки.
*   **Просмотр маршрутов**: На главной странице отображается список всех созданных маршрутов.
*   **Детали маршрута**: У каждого маршрута есть своя страница, на которой отображаются:
    *   Интерактивная карта с метками для всех мест в маршруте.
    *   Список мест (достопримечательности, рестораны и т.д.).
    *   Рекомендации от AI по добавлению новых мест в маршрут.

### 2. Рекомендации и поиск
*   **AI-рекомендации**: Приложение использует AI для подбора туров, жилья, ресторанов, транспорта и развлечений на основе предпочтений пользователя.
*   **Мультифильтр**: Страница с возможностью одновременной фильтрации по нескольким категориям (туры, жилье, и т.д.).
*   **Фильтр-карта**: Интерактивная карта для визуального поиска и фильтрации предложений, позволяющая сравнивать варианты из разных категорий в реальном времени.

### 3. Управление бизнесом (для владельцев)
*   **Регистрация бизнеса**: Отдельный флоу регистрации для владельцев бизнеса.
*   **Панель управления**: Специальный раздел, где владельцы могут просматривать свои предложения в виде доски объявлений.
*   **Добавление предложений**: Форма для создания новых бизнес-предложений (туров, отелей, ресторанов) с указанием названия, описания, категории, местоположения и URL изображения.

### 4. Пользовательский профиль и аутентификация
*   **Аутентификация**: Пользователи могут регистрироваться и входить в систему с помощью Email/пароля. Реализовано через Firebase Authentication.
*   **Разделение ролей**: Система различает обычных пользователей (путешественников) и владельцев бизнеса.
*   **Личный кабинет**: Пользователи могут просматривать и редактировать свою информацию.
*   **Избранное и Заказы**: В профиле путешественника есть разделы для отслеживания понравившихся предложений и истории бронирований.
*   **Уведомления**: Страница с уведомлениями об обновлениях и специальных предложениях.

## Как запустить приложение

Чтобы запустить сервер для разработки, выполните следующую команду в вашем терминале:
```bash
npm run dev
```
Приложение будет доступно по адресу `http://localhost:9002`.

## Структура проекта

*   `src/app/` — Основной каталог со страницами приложения.
    *   `dashboard/` — Страницы панели управления для владельцев бизнеса.
    *   `profile/` — Страница профиля пользователя.
    *   `tours/`, `housing/`, `restaurants/` и т.д. — Страницы категорий.
*   `src/components/` — Переиспользуемые React-компоненты.
    *   `pages/` — Компоненты, составляющие основной контент страниц.
    *   `ui/` — Компоненты из библиотеки ShadCN UI.
*   `src/ai/` — Логика для работы с искусственным интеллектом (Genkit).
*   `src/firebase/` — Конфигурация и хуки для работы с Firebase.
*   `src/lib/` — Вспомогательные функции, утилиты и данные.
*   `docs/` — Документация по проекту.
    * `backend.json` - Описание структуры данных для Firebase.
