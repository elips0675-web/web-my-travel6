# Приложение "Путевой Компас"

## Обзор

Это приложение на Next.js, разработанное как AI-помощник для планирования путешествий. Оно помогает пользователям создавать маршруты поездок и получать рекомендации по турам, жилью, аренде авто и развлечениям от искусственного интеллекта.

## Технологический стек

*   **Фреймворк**: Next.js (с App Router)
*   **Язык**: TypeScript
*   **UI**: React
*   **Стилизация**: Tailwind CSS
*   **UI Компоненты**: ShadCN UI
*   **AI**: Google Genkit
*   **Иконки**: Lucide React
*   **Карты**: Google Maps React (@vis.gl/react-google-maps)
*   **Формы**: React Hook Form и Zod
*   **PWA**: Next-PWA (@ducanh2912/next-pwa)

## Настройка окружения

Для полноценной работы приложения требуется настроить переменные окружения. Эти переменные содержат секретные ключи для подключения к внешним сервисам и не должны храниться в системе контроля версий.

1.  **Создайте файл `.env.local`**: В корневой директории проекта создайте файл с именем `.env.local`.

2.  **Добавьте ключи**: Скопируйте в него следующее содержимое и замените заполнители `YOUR_..._HERE` на ваши реальные ключи.

    ```
    # Ключи для Firebase
    NEXT_PUBLIC_FIREBASE_API_KEY=YOUR_API_KEY_HERE
    NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=YOUR_AUTH_DOMAIN_HERE
    NEXT_PUBLIC_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID_HERE
    NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=YOUR_STORAGE_BUCKET_HERE
    NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=YOUR_MESSAGING_SENDER_ID_HERE
    NEXT_PUBLIC_FIREBASE_APP_ID=YOUR_APP_ID_HERE

    # Ключ для Google AI
    GOOGLE_GENAI_API_KEY=YOUR_GOOGLE_AI_API_KEY_HERE
    ```

### Где взять ключи?

*   **Ключи Firebase**:
    1.  Перейдите в [консоль Firebase](https://console.firebase.google.com).
    2.  Выберите ваш проект.
    3.  Нажмите на иконку шестеренки (⚙️) и выберите **"Настройки проекта" (Project settings)**.
    4.  На вкладке "Общие" прокрутите до раздела "Ваши приложения".
    5.  В конфигурации вашего веб-приложения (объект `firebaseConfig`) вы найдете все необходимые ключи.

*   **Ключ Google AI**:
    1.  Перейдите в [Google AI Studio](https://aistudio.google.com/app/apikey).
    2.  Создайте новый API-ключ.

После того как вы заполните файл `.env.local` реальными данными, необходимо **перезапустить сервер для разработки**, чтобы изменения вступили в силу.

## Как запустить приложение

Чтобы запустить сервер для разработки, выполните следующую команду в вашем терминале:
```bash
npm run dev
```
Приложение будет доступно по адресу `http://localhost:9002`.

## Функционал

### 1. Управление маршрутами
*   **Создание маршрутов**: Пользователи могут создавать новые маршруты путешествий, указывая название, пункт назначения и даты поездки.
*   **Просмотр маршрутов**: На главной странице отображается список всех созданных маршрутов.
*   **Детали маршрута**: У каждого маршрута есть своя страница, на которой отображаются:
    *   Интерактивная карта с метками для всех мест в маршруте.
    *   Список мест (достопримечательности, рестораны и т.д.).
    *   Рекомендации от AI по добавлению новых мест в маршрут.

### 2. Рекомендации от AI
*   **Подбор туров**: На странице "Туры" пользователи могут найти туры и экскурсии на основе пункта назначения, дат и интересов.
*   **Подбор жилья**: Страница "Жилье" позволяет пользователям находить жилье (отели, апартаменты) на основе пункта назначения, дат и предпочтений.
*   **Подбор ресторанов**: Страница "Кафе и рестораны" помогает найти подходящие заведения на основе предпочтений в еде и атмосфере.
*   **Подбор транспорта**: На странице "Транспорт" пользователи могут находить различные виды транспорта: каршеринг, такси, велосипеды или самокаты. AI предлагает подходящие варианты на основе местоположения, дат и предпочтений.
*   **Подбор развлечений**: Страница "Развлечения" помогает найти активности, такие как квесты, VR-арены, боулинг и т.д.
*   **Фильтр-карта**: Интерактивная карта для визуального поиска и фильтрации предложений по разным категориям. Позволяет находить и сравнивать варианты из разных категорий (туры, жилье, развлечения и т.д.) одновременно, с обновлением результатов в реальном времени.

### 3. Профиль пользователя и Панель управления
*   **Современный дизайн профиля**: Страница профиля полностью переработана, чтобы предложить пользователям более чистый, интуитивно понятный и современный интерфейс.
*   **Демо-режим для неавторизованных пользователей**: Пользователи без авторизации могут исследовать функционал приложения через демонстрационные режимы:
    *   **Профиль**: Возможность переключаться между демо-профилями "Путешественник" и "Владелец бизнеса", чтобы увидеть, как они выглядят.
    *   **Уведомления**: Просмотр демонстрационных уведомлений, распределенных по категориям: "Все", "Бронирования", "Избранное" и "Бизнес".
*   **Панель управления (Dashboard)**: Владельцы бизнеса могут управлять своими объектами и услугами. Для неавторизованных пользователей доступен демо-режим с примерами карточек для различных категорий бизнеса (туры, жилье, рестораны и т.д.).

### 4. Пользовательский интерфейс
*   **Адаптивный дизайн**: Приложение корректно отображается как на настольных компьютерах, так и на мобильных устройствах.
*   **Современный UI**: Создан с использованием компонентов ShadCN UI для чистого и современного внешнего вида.
*   **Удобная навигация**: Главное меню обеспечивает быстрый доступ ко всем основным разделам.

### 5. Progressive Web App (PWA)
*   **Установка на устройство**: Пользователи могут установить приложение на свой смартфон или компьютер для быстрого доступа, как нативное приложение.
*   **Офлайн-доступ**: Базовый доступ к приложению сохраняется даже без подключения к интернету.
*   **Баннер установки**: На мобильных устройствах появляется удобный баннер, предлагающий установить приложение.

## Структура проекта

Вот краткий обзор основных каталогов и файлов:

*   `src/app/` — Основной каталог со страницами приложения, построенный на Next.js App Router.
    *   `layout.tsx` — Корневой макет приложения.
    *   `page.tsx` — Главная страница.
    *   `tours/page.tsx` — Страница "Туры".
    *   `housing/page.tsx` — Страница "Жилье".
    *   `restaurants/page.tsx` — Страница "Кафе и рестораны".
    *   `activities/page.tsx` — Страница "Развлечения".
    *   `rental-car/page.tsx` — Страница "Транспорт".
    *   `filter-map/page.tsx` — Страница с фильтром на карте.
    *   `routes/[id]/page.tsx` — Динамическая страница для просмотра деталей маршрута.
    *   `tours/[slug]/page.tsx`, `housing/[slug]/page.tsx` и т.д. — Динамические страницы для деталей.
*   `src/components/` — Переиспользуемые React-компоненты.
    *   `ui/` — Компоненты из библиотеки ShadCN UI.
    *   `pages/` — Компоненты, составляющие контент страниц.
    *   `map.tsx` — Компонент для отображения карты Google.
*   `src/ai/` — Логика для работы с искусственным интеллектом (Genkit).
    *   `flows/` — Файлы с AI-потоками для генерации рекомендаций.
*   `src/lib/` — Вспомогательные функции, утилиты и данные.
    *   `data.ts` — Временные данные для маршрутов (пока нет базы данных).
    *   `utils.ts` — Утилиты, например, для объединения CSS-классов.
*   `docs/` — Документация по проекту.
*   `package.json` — Список зависимостей и скриптов проекта.
*   `tailwind.config.ts` — Файл конфигурации Tailwind CSS.
*   `next.config.ts` — Файл конфигурации Next.js.
